sequa is a generative drum machine and synthesis engine with a pyqt6 interface.	  

it has 808-style synthesis with polyrhythmic sequencing, along with physics-based parameter drifting.	  

**sliders**  
*horizontal/all*
- bpm  
- swing  
- clip  
- evolve
*vertical/individual*
- bit (downsample)  
- flt (lowpass/highpass)  
- pch (pitch)  
- dec (decay)  
- ton (tone)
