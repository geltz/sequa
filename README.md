sequa is a generative drum machine and synthesis engine with a pyqt6 interface.	  

it has 808-style synthesis with polyrhythmic sequencing, along with physics-based parameter drifting.	  

it also lets you import your own sounds by dragging an audio file on any pad title.    

**buttons**
- wav (save sound)    
- vel (rand velocity)  
- rnd (rand velocity + position)
- export loop
- clear all

**sliders**  

all
- bpm  
- swing  
- clip  
- evolve

one
- bit (downsample)  
- flt (lowpass/highpass)  
- pch (pitch)  
- dec (decay)  
- ton (tone)
