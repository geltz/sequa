sequa is a generative drum machine and synthesis engine.	    
it lets you import your own sounds in-place by clicking/dragging on pad titles.    

**controls**

left/right mouse to add/remove pads and adjust velocity.  
click the top left logo to swap kits.

**kits**	
	
- eight		
- nine		
- simple		
- pcm		

**buttons**

| button | action |
| :--- | :--- |
| wav | save drum |
| vel | random velocity |
| rnd | random velocity/position |
| low | lower velocity |
| export loop | export audio loop |
| clear all | clear pattern |

**sliders**  

| slider | effect |
| :--- | :--- |
| bpm | set tempo |
| swing | add swing |
| cut | cut |
| evo | evolve |
| rev | reverse |
| bit | bitcrush |
| flt | filter |
| pch | pitch |
| dec | decay |
| ton | tone |

**reseq**

| feature | description |
| :--- | :--- |
| snp | random snap |
| rev | reverse |
| flt | random filters |

| del | random delays/reverb |
