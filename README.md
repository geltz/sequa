sequa is a generative drum machine and synthesis engine.	    

it lets you import your own sounds in-place by clicking/dragging on pad titles.    

**controls**

left/right mouse to add/remove pads and adjust velocity.  
click the top left logo to swap kits.

**kits**	
	
- eight		
- nine		
- simple		
- pcm		

**buttons**

- wav → save drum	    
- vel  → velocity  
- rnd → velocity/position
- export loop
- clear all

**sliders**  

- bpm  
- swing  
- cut  
- evo → evolve  
- rev → reverse
- bit → bitcrush   
- flt → filter  
- pch → pitch  
- dec → decay  
- ton → tone	

**special**

reseq → file cut-up sequence
- snp → random snap
- rev → reverse
- flt → random filters
- del → random delays/reverb
